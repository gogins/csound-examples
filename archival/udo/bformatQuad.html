bformatQuad

Converts quadrophonic to b-format.

Download UDO File

Description

Converts quadrophonic to b-format.

Syntax

aW, aX, aY, aZ bformatQuad aFL, aFR, aRL, aRR

Code

opcode bformatQuad, aaaa, aaaa
	aFL, aFR, aRL, aRR xin

	; ignore z channel, as no height information in quad
	aW     =  aFL + aFR + aRL + aRR
	aX     =  aFL + aFR - aRL - aRR
	aY     =  aFL - aFR + aRL - aRR
	aZ     =  0   

	xout aW, aX, aY, aZ
endop
