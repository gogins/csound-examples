User-Defined Opcode Database

declick

removes clicks from signal start and end

Download UDO File

Description

Given a signal, it will DeClick will apply an envelope to it to ensure there is no clicking at the start or end of the sound

(Taken from the Csound Reference Manual)

Syntax

aout declick ain

Performance

ain - signal to declick
aout - declicked signal

Code

        opcode declick, a, a

ain     xin
aenv    linseg 0, 0.02, 1, p3 - 0.05, 1, 0.02, 0, 0.01, 0
        xout ain * aenv         ; apply envelope and write output

        endop
