About the User-Defined Opcode Database

This is a repository of User-Defined Opcodes for the Csound music synthesis software. User-Defined Opcodes are opcodes for Csound created using Csound orchestra text. More information about User-Defined Opcodes can be found here.

Using the database to browse and download UDO's does not require registration or logging in.

Registration allows users to login and add UDO's, as well as edit or remove those which they have added.

This site also includes an RSS feed so end users can monitor when new UDO's have been added. (Currently only shows new entries, not modified ones).
